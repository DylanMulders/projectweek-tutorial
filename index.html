HTML5 semantische elementen
header - Definieert een koptekst voor een document of een sectie
nav - Definieert een container voor navigatielinks
section - Definieert een sectie in een document
article - Definieert een onafhankelijk, op zichzelf staand artikel
aside - Definieert inhoud naast de inhoud (zoals een zijbalk)
footer - Definieert een voettekst voor een document of een sectie
details - Definieert aanvullende details

CSS Lay-out



Koptekst.
Een koptekst bevindt zich meestal bovenaan de website (of direct onder een navigatiemenu bovenaan). Het bevat vaak een logo of de naam van de website .header {
background-color: #F1F1F1;
text-align: center;
padding: 20px; }
Een navigatiebalk bevat een lijst met links waarmee bezoekers door uw website kunnen navigeren

/* The navbar container /
.topnav {
overflow: hidden;
background-color: #333;
}
/ Navbar links /
.topnav a {
float: left;
display: block;
color: #f2f2f2;
text-align: center;
padding: 14px 16px;
text-decoration: none;
}
/ Links - change color on hover */
.topnav a:hover {
background-color: #ddd;
color: black;
}
Inhoud De lay-out in dit gedeelte is vaak afhankelijk van de doelgebruikers. De meest voorkomende lay-out is een (of een combinatie van) van de volgende:
1-kolom (vaak gebruikt voor mobiele browsers)
2-koloms (vaak gebruikt voor tablets en laptops)
Indeling met 3 kolommen (alleen gebruikt voor desktops)

/* Create three equal columns that floats next to each other /
.column {
float: left;
width: 33.33%;
}

/ Clear floats after the columns /
.row:after {
content: "";
display: table;
clear: both;
}

/ Responsive layout - makes the three columns stack on top of each other instead of next to each other on smaller screens (600px wide or less) */
@media screen and (max-width: 600px) {
.column {
width: 100%;
} }
